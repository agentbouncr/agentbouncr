## Reporting a Vulnerability

Please report security vulnerabilities to **info@agentbouncr.com**.

- We will acknowledge receipt within 48 hours
- We will provide an initial assessment within 5 business days
- We will work with you to understand and resolve the issue

Please do not open public GitHub issues for security vulnerabilities.

## Responsible Disclosure

We ask that you give us reasonable time to address vulnerabilities
before disclosing them publicly. We are committed to working with
security researchers to resolve issues promptly.
