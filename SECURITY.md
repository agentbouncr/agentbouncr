## Reporting a Vulnerability

Please report security vulnerabilities to **info@agentbouncr.com**.

- We will acknowledge receipt within 72 hours
- We will provide an initial assessment within 7 business days
- We will work with you to understand and resolve the issue

**Do not open public GitHub issues for security vulnerabilities.**

## Responsible Disclosure

We ask that you give us reasonable time to address vulnerabilities before public disclosure. We are committed to working with security researchers to resolve issues promptly.
